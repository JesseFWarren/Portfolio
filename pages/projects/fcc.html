Federal Communications Commission Data Analytics Internship

As a Data Aanalytics intern at the FCC I worked with the national alert data from the FEMA database using the IPAWS API.

This project had 3 main steps:
1. Develop API functions to filter json data for date/time ranges, data types, etc. and download that data.
2. Streamline a process to convert json and xml data into a readable csv fiormat and create a single python script to download and convert the data from the FEMA database.
3. Deploy a Dashboard using PowerBI to analyze csv data for FCC employees to have a more efficient way to analyze and gain insights from the FEMA Alert data.
Additionally, develop user manuals and running documentation allowing the project to easily be maintained after I end my internship.

I worked in the Public Safety and Homeland Security Bureau under a supervisor. We utilized the agile method with weekly sprint reports and meetings. This internship taught me a lot about how the Federal Government opperates and I am very thankful I had the oppurtunity to make many meaningful connections during my time at the FCC.
